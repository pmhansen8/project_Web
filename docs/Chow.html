Choice

This project is a game where users search for places to eat

📌 Features

  • Geo location
  • AI recommendations
  • User authentication and authorization
  • Recommendation saving

🛠️ Technologies Used

  • JavaScript
  • Open AI API
  • Fire Base
  • React

🚀 How It Works

The user makes an account and is able type in what they want to eat and get recommendations.
