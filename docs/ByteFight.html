Byte Fight AI

This project is a game where users make AI snake bots to fight against others

📌 Features

  • leader board
  • game engine functionality
  • User authentication and authorization
  • game state and elo tracking

🛠️ Technologies Used

  • JavaScript
  • Java
  • Spring boot
  • Sql

🚀 How It Works

The user makes an account and is able to edit a file given. after doing so they are then able to download the file to the game engine which in turn will run the file.
