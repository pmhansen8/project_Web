Guess The Price

This project is a game where users guess the price of different items

📌 Features

  • leader board
  • lazy loading
  • User authentication and authorization
  • score caching

🛠️ Technologies Used

  • JavaScript
  • Java
  • Node
  • Firebase

🚀 How It Works

The user makes an account and is able to guess the prices of items. The score of that user is saved.
